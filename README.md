# 1、Windows系统
# 2、python3.0+
# 3、一个美丽帅气的自己

# 1、脚本执行会自动保存一个备份文件，如不放心在执行前可自行备份。
# 2、文件是C:/Windows/System32/drivers/etc/这个目录下的hosts文件
# 3、文件会多生成两个文件，hosts+时间.bak文件是原始的hosts文件，hosts.bak文件为上次修改的文件
